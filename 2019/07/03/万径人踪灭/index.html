万径人踪灭

发表于 2019-07-03 更新于 2021-06-17 分类于 OI 阅读次数：

万径人踪灭 搜了下千山鸟飞绝这题也有，挺整齐的= = 求不相邻的回文串个数，就是总回文串数减去相邻数。相邻的manacher随便做。 对于一个对称重心$i$，存在$x$对字母关于它对称，如果$i$在一个字母位，可以形成的回文串数就是$2^{x+1}-1$。如果$i$在两个字母中间，就是$2^x-1$。 因为这题只有两个字母，分开考虑各有多少对。，把考虑的字母设为$1$，其它设为$0$，卷一下就好了。我 卷 我 自 己
