TJOI2017DNA

发表于 2019-07-02 更新于 2021-06-17 分类于 OI

[TJOI2017]DNA 热动分析了一坨解法：

解法1：

SA，因为最多能失配三次嘛，就枚举起点，用SA求LCP，跳三次以内匹配完就算可以。 没代码。

解法2：

SAM，先算一下每个点的size，然后直接暴力dfs。记录一下失配边数，搜到3结束。 没代码。

解法3：

因为只有四个字母，枚举字母，把A串有的地方记为1，B串没有的地方记为1。两个串卷一下，1的个数就是本字母不一样的个数。跑的挺慢，NTT能勉强卡过去。
