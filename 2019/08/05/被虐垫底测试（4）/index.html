被虐垫底测试（4）

发表于 2019-08-05 更新于 2021-06-17 分类于 OI

题目在这里 花了一个多小时rush了$284$分，没想到炸成了$300-284$分，自闭了。。。

T1

最水的题。考虑到乘法顺序无关，可以先把行的算完，再算列的。进一步观察，每一列都可以表示为第一列加一个数（等差数列）。这就好办了。

T2

用线段树维护，每个节点维护一个数组，$nxt[i]$表示$x$为$i$，走过这一段是什么（其实就是置换） 置换明显是满足结合律的，可以矩乘。 这题放一下代码吧。

T3

$84$分做法： 询问的是$[l,r]$，那么就找到$[l,r]$中的最大最小值。再找到这两个值之间所有值，出现位置最左最右的位置。循环，直到不再变化。 $100$分做法： 点这里。 [CERC2017]Intrinsic Interval [Tarjan][线段树建图]
