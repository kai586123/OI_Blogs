ZJOI2015诸神眷顾的幻想乡

发表于 2019-06-18 更新于 2021-06-17 分类于 OI

[ZJOI2015]诸神眷顾的幻想乡 15年某集训队论文里钦定的题。 树上一端路径，一定可以看作一个点为根向下走到某个叶子，这条路径上的其中一段。题目有个非常强的条件：叶子不超过20个，即线性的。 枚举叶子为根，dfs整个树，把树当作trie，插进SAM，就是个广义SAM。其实貌似这样点比较多，不一定线性，bfs才行，不过貌似没人卡= = 不同子串数量，就是len[x]-len[fa[x]]。
