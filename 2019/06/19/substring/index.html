SubString

发表于 2019-06-19 更新于 2021-06-17 分类于 OI

SubString CLJ课件里钦定的题，做法很简单：出现次数就是SAM上parent树的子树和。动态维护parent树，使用LCT。每次插入新点的时候，把这个点到根全+1。 然后代码调好久… 一些奇怪的代码注意点： 本题维护的是有根，有向的树，而且link和cut操作保证合法，所以不需要makeroot。 因为是区间+1，要打LazyTag。 回答询问之前，先preview一下，从根pushdown到这个点。
