NOIP2018 摆渡车 斜率优化DP

发表于 2019-09-16 更新于 2021-06-17 分类于 OI

摆渡车 斜率优化这玩意一直都是嘴巴过题，基本没写过，于是PJ水题居然调不出来。。。。。。丢死人 DP式子很简单。$f_i$为到时间$i$答案，令$c_i,s_i$分别为到$i$有的人数，人时间前缀和。 $$f_i=\min_{j=1}^{i-m} f_j+(c_i-c_j)i + (s_i-s_j)$$ 斜率优化一下，令$j < k, f_j > f_k$ $$i > \frac{(f_k+s_k)-(f_j+s_j)}{c_k-c_j} $$ 维护下凸包即可。 然后就是代码的坑了。。。

  1. 可能$c_k-c_j$相同。此时，比较两点$y$的大小
  2. 可以转移来的状态是$i-m$来的
  3. $[1,m]$的$f$需要预处理。然后，从$m$开始向后计算，因为$f_{m-m}=f_0$可以使用。
